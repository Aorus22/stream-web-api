## Embedded static server (Go)

This small Go program embeds the `dist/` directory (generated by your frontend build) into a single executable and serves it as a static SPA.

Build & run:

1. Build the frontend:

   ```bash
   npm run build
   # or pnpm build / yarn build depending on your setup
   ```

2. Build the Go executable (embedding the current `dist/`):

   ```bash
   go build -o stream-web-server .
   ```

3. Run it:

   ```bash
   ./stream-web-server -addr ":8080"
   ```

Notes:
- Whenever you change the frontend, re-run the frontend build then re-build the Go binary to update embedded assets.
- The server provides an SPA fallback: unknown paths return `index.html` so client-side routing works.
